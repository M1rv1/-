-# Программа расписания аничкого лицея
1) txt файл с окнами и excel файл с распианием будут созданы в дирекотории с самой программой
3) Чтобы изменять количетство классов в расписании, стоит просто комментировать их
4) На строчке 2467 после знака "=" можно выбрать количество секунд для перебора перебора вариантов
5) Данные о нагрузке и о доплнительных уроках в школе прилагаются в этой ветке
6) Исключения самих учителей выписаны из приватной таблицы, доступ к которой невозможен
