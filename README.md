-# Программа расписания аничкого лицея
1) Чтобы изменять количетство классов в рассаписании стоит просто комментировать их
2) На строчке 2467 после знакак равно можно выбрать количество секунд перебора вариантов
3) Данные о нагрузке и о доплнительных уроках прилагаются в этой ветке
4) Исключения самих учителей выписаны из приватной таблицы, доступ к которой невозможен
