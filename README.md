-# Программа расписания аничкого лицея
1) Чтобы изменять количетство классов в расписании, стоит просто комментировать их
2) На строчке 2467 после знака "=" можно выбрать количество секунд для перебора перебора вариантов
3) Данные о нагрузке и о доплнительных уроках в школе прилагаются в этой ветке
4) Исключения самих учителей выписаны из приватной таблицы, доступ к которой невозможен
